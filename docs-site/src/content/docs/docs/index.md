Suitest is a self-hostable, open-source testing platform that works **with or
without an LLM**:

- **ZERO tier** — manual test case management + a deterministic run engine that
  drives any target through [MCP](https://modelcontextprotocol.io) (Playwright,
  HTTP, Postgres, and more). No LLM, ever.
- **LOCAL tier** — plug in a local model (Ollama, vLLM, llama.cpp, LM Studio) for
  AI generation, diagnosis, and chat — fully air-gappable.
- **CLOUD tier** — bring your own cloud LLM key (Anthropic, OpenAI, …).

Start with [Getting started](/docs/guides/getting-started/).
