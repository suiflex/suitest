# Suitest examples (M4-15)

Runnable starting points for common Suitest workflows. Each directory has its own
README with step-by-step instructions.

| Example | What it shows | Tier |
|---------|---------------|------|
| [`demo-app`](./demo-app) | Full sample app (FastAPI + static UI) with a ready-to-run Suitest suite | ZERO |

The demo app runs at ZERO tier (no LLM). Set `SUITEST_API_URL` / `SUITEST_TOKEN` /
`SUITEST_WORKSPACE_ID` and use the `suitest` CLI or the SDKs.
